## Introdução

Este aplicativo foi desenvolvido como parte de um desafio técnico e tem como objetivo principal permitir a leitura de e-books no formato EPUB.

### Recursos

- **Leitura de E-books:** Permite abrir e ler e-books no formato EPUB.
- **Marcação de Favoritos:** Capacidade de marcar livros como favoritos para acesso rápido.

## Iniciando

### Pré-requisitos

Para executar o aplicativo localmente, você precisará ter instalado:

- Flutter SDK
- Dispositivo Android/iOS ou Emulador

### Instalação

1 - Clone o repositório:

```bash
git clone https://github.com/FiilipeRibeiro/LeitorDeEbooks.git
```

2 - Navegue até o diretório do projeto:

```bash
cd LeitorDeEbooks
```

3 - Instale as dependências:

```bash
flutter pub get
```

Executando 

Para executar o aplicativo, use o comando:

```bash
flutter run
```

Isso iniciará o aplicativo no dispositivo ou emulador conectado.
